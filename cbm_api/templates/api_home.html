Predict Performance Degradation State of Naval Propulsion Plants

Introduction

This API predicts the performance degradation state of naval propulsion plants given the following 2 features:
  1. Gas Turbine shaft torque (GTT) [KNm] ( Type: Continuous )
  2. Gas Generator rate of revolutions (GGn) [rpm] ( Type: Continuous )

How to Use

Below is an example, in python file, of how to use this API: suppose there is a vessel which has a Gas Turbine shaft torque of 12 KNm and a gas generator of revolution of 2 rpm, as shown in Table 1.0 below, then, the performance degradation state can be predicted as shown below:

Table 1.0: Sample Data Point
Gas Turbine shaft torque (GTT) [kNm] Gas Generator rate of revolutions (GGn) [rpm]
12 2

The API can be used in a python file (named client.py in this case) as shown below

client.py
            import requests

            endpoint = "http://127.0.0.1:8000/cbm_api"
            data_dict = {
                'Gas Turbine shaft torque (GTT) [kN m]': 12,
                'Gas Generator rate of revolutions (GGn) [rpm]': 2
            }

            predict = requests.post(endpoint, json=data_dict)

            performance_degradation_state_dict = predict.json()
            print("Performance Degradation State Dictionary = ",  performance_degradation_state_dict)

            performance_degradation_state = performance_degradation_state_dict['performance_degradation_state']
            print("Performance Degradation State = ", performance_degradation_state)
